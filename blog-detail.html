Lifelong learning is a form of self-initiated education that is focused on personal development. it has generally been taken to refer to the learning that occurs outside of a formal educational institute, such as a school, university or corporate training.

Responsibility
  • Information management system for displaying on website pages, such as banner images, public relations banners, categories, tags, course overview, course status, etc.
  • User role management system that can access the information management system
  • Dashboard showing usage data, such as the number of active courses by category, total number of courses by activation status, courses with the top 10 students who have attended the course.
  • Debit/Credit card and QR payment system using payment gateway provider.
Language & Framework
  • Moodle (LMS)
  • HTML
  • CSS
  • PHP
  • AngularJs
  • MySQL

e-Learning is Learning Management System (LMS), System that focus on the students as the center of teaching and learning in combination with traditional place-based classroom learning. And can be used as a full online learning management system. Providing a system for faculty, students, and personnel of university.

Responsibility
  • User Interface: Develop theme for moodle.
  • Data Management System: Develop system allows for faster management or editing of large amounts of data.
  • Calculation system: Develop service cost calculation system to provide monthly reports.
  • Online examination booking system. Develop a booking system so that the system can be used without problems when there are many users at the same time.

  • Moodle (LMS)

  • JavaScript
